# CrowdSense Documentation

Welcome to the CrowdSense project! This documentation provides an overview of the project structure, setup, and contribution guidelines.

## Project Structure
- `core/`: Core logic and modules
- `simulation/`: Simulation tools and scripts
- `utils/`: Utility functions and helpers
- `web/`: Web application code
- `tests/`: Unit and integration tests

## Getting Started
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run tests: `python -m unittest discover tests`
